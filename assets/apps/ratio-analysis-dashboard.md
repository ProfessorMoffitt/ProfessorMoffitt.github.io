# Ratio Analysis Dashboard

Enter simplified financial statement amounts to calculate common liquidity, profitability, solvency, and efficiency ratios. Use the results to practice interpreting financial performance and position.

This mini-app is designed for accounting students who are learning how to analyze financial statements. Students can enter income statement and balance sheet amounts, calculate common ratios, and read short explanations of what each ratio means.

[Open the Ratio Analysis Dashboard]({{ site.baseurl }}/assets/apps/ratio-analysis-dashboard/)

## What the app calculates

The dashboard includes several common financial statement ratios:

### Liquidity ratios

- Current ratio
- Quick ratio

### Profitability ratios

- Gross margin
- Operating margin
- Net profit margin
- Return on assets
- Return on equity

### Solvency ratios

- Debt-to-equity ratio
- Debt ratio
- Times interest earned

### Efficiency ratios

- Inventory turnover
- Receivables turnover
- Asset turnover

## How to use it

1. Enter simplified financial statement amounts.
2. Leave fields blank if you do not have the data.
3. Click **Calculate Ratios**.
4. Review the ratio value, formula, interpretation, and student note.
5. Use **Load Sample Company Data** to test the app quickly.
6. Use **Clear Inputs** to start over.

## Educational note

This tool is for educational purposes only. Ratio interpretation depends on industry, company strategy, accounting policies, and other context.
